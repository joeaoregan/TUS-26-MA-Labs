### Setup 

#### Virtual Environment

```bash
python -m venv venv
.\venv\Scripts\Activate.ps1
```

#### Install Material for MkDocs

```bash
pip install mkdocs-material
```
or
```bash
python -m pip install mkdocs-material
```

Create a new site in the current folder:

```bash
python -m mkdocs new .
```
